# MySearch —— 本地文档索引与搜索工具

一个基于 Python 的本地文档搜索系统，支持对文件夹内多种类型文档进行索引构建，并通过关键词进行相关性排序搜索，提供命令行版本与可视化界面版本。

---

## 功能简介

- 扫描指定文件夹并自动构建文档索引
- 支持中文关键词搜索（基于 TF-IDF 相关性排序）
- 使用结巴分词（jieba）进行中文分词
- 支持多种文件格式：
  - `.txt` / `.md` / `.py`
  - `.pdf`
  - `.docx`（可选）
  - `.pptx`（可选）
  - `.csv` / `.xlsx`（可选）
  - `.html`
- 索引结果本地持久化，避免重复构建
- 提供可视化界面，支持点击结果直接打开原文件

---

## 项目结构说明

```text
.
├── 可视化.py        # 图形界面版本（主程序）
├── 搜索.py          # 简化版搜索引擎示例（内置文档）
├── 遍历.py          # 文件夹扫描 + 命令行搜索版本
├── stopwords.txt    # 中文停用词表
├── indexes/         # 本地生成的索引文件（运行时自动创建）
└── README.md
```

---

### 搜索引擎核心（RankedSearchEngine）

三个程序均基于同一套搜索引擎逻辑，主要特点：

- 使用 TF-IDF 模型计算文档相关性
- 词频（TF） + 文档频率（IDF）组合评分
- 支持停用词过滤
- 搜索结果按相关性降序排序

---

## 遍历.py —— 命令行搜索版本

**功能：**
- 扫描指定文件夹
- 自动读取并解析文档内容
- 构建索引并在终端进行交互式搜索

**适合：**
- 理解搜索引擎原理
- 课程实验 / 控制台演示

**运行方式：**
```bash
python 遍历.py
```

运行后可在命令行中输入关键词进行搜索。

---

## 搜索.py —— 搜索引擎简化示例

**功能：**
- 内置少量示例文本
- 演示 TF-IDF 搜索与高亮显示

**适合：**
- 学习和测试搜索算法
- 理解评分逻辑

**运行方式：**
```bash
python 搜索.py
```

---

## 可视化.py —— 图形界面搜索系统（主程序）

**功能：**
- 基于 customtkinter 构建 GUI
- 支持文件夹选择与历史记录
- 索引自动保存与加载
- 搜索结果卡片式展示
- 关键词高亮
- 点击结果直接打开原文件

**运行方式：**
```bash
python 可视化.py
```

或在 Releases 页面下载打包后的 `可视化.exe` 直接运行。

## 使用说明

1. 运行程序并选择需要索引的文件夹
2. 点击“重建索引”生成文档索引
3. 输入关键词进行搜索
4. 点击搜索结果即可打开对应文件
