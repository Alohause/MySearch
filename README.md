# MySearch —— 本地文档索引与搜索工具

一个基于 Python 的本地文档搜索系统，支持对文件夹内多种类型文档进行索引构建，并通过关键词进行相关性排序搜索，提供命令行版本与可视化界面版本。

---

## 功能简介

- 扫描指定文件夹并自动构建文档索引
- 支持中文关键词搜索（基于 TF-IDF 相关性排序）
- 使用结巴分词（jieba）进行中文分词
- 支持多种文件格式：
  - `.txt` / `.md` / `.py`
  - `.pdf`
  - `.docx`（可选）
  - `.pptx`（可选）
  - `.csv` / `.xlsx`（可选）
  - `.html`
- 索引结果本地持久化，避免重复构建
- 提供可视化界面，支持点击结果直接打开原文件
